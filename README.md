# atv-integradora-modulo-gerenciamento-pouso

# 📋 Módulos da Base Aurora Siger — Inspirados no NASA Base Camp

O base camp é dividido em **4 zonas operacionais** e os módulos são entregues em **3 fases**.
Cada elemento que chega via lander compõe a fila de pouso gerenciada pelo **MGPEB**.

---

## 🏠 Zona de Habitação (Habitation Zone)

| Módulo | Sigla/Nome NASA | Função |
|--------|----------------|--------|
| Habitat Principal | Common Habitat | Habitação para 8 tripulantes, laboratórios, medicina, comando |
| Nó de Airlock Duplo | TCAN | Câmara de pressurização para EVA |
| Módulo de Logística | LM (×2) | Depósito de suprimentos (vira módulo de lixo depois) |
| Rover Pressurizado | PR (×2) | Transporte interno pressurizado da equipe |
| Pallet de Utilidade | PUP (×2, acoplado nos rovers) | Carregamento auxiliar nos rovers |
| Campo de Radiadores | Radiator Field | Rejeição de calor + torre de comunicações |

---

## 🛬 Zona de Pouso (Landing Zone)

| Módulo | Sigla/Nome NASA | Função |
|--------|----------------|--------|
| Terminal de Energia | Power Terminal | Fornece energia para landers na superfície via cabo |
| Carrinho de Cabos | Cable Cart | Conecta o terminal ao lander (até 200m de cabo) |
| Rover de Terreno (Marte) | MTV (×2) | Mobilidade no entorno da zona de pouso |

---

## ⚡ Zona de Energia (Power Zone)

| Módulo | Sigla/Nome NASA | Função |
|--------|----------------|--------|
| Reator Nuclear de Fissão | Nuclear Power Element | Fonte primária de energia da base toda |
| Radiadores do Reator | Radiators | Dissipação térmica do reator |
| Array de Comunicações Externo | External Comms Array | Comunicação superfície–espaço |
| Terminal de Distribuição | Power Distribution Terminal | Distribui energia para todas as zonas via cabos enterrados |

---

## 🔬 Zona de Produção de Recursos (Resource Production Zone)

| Módulo | Sigla/Nome NASA | Função |
|--------|----------------|--------|
| Planta ISRU | ISRU Payload (×4) | Produção de propelente, água e oxigênio |
| Robôs Escavadores | RASSOR (×3) | Escavação de regolito |
| Robôs de Transporte Hexápodes | ATHLETE (×8 no total) | Transporte de módulos pesados, escavação |
| Lâminas Bulldozer | LANCE (×3) | Preparação de terreno |
| Tanque de LOX (O₂ líquido) | LOX Tanker | Transporte de oxigênio produzido |
| Tanque de Metano | Methane Tanker | Transporte de metano (combustível ISRU) |
| Tanque de Água | Water Tanker | Transporte de água extraída |

## 🏗️ Equipamentos de Construção

| Módulo | Sigla/Nome NASA | Função |
|--------|----------------|--------|
| Guindaste de Superfície | LSMS (×3) | Içamento e posicionamento de módulos pesados |
